 >> O que é GIT ???
GIT é um sistema de versionamento de código, que guarda os registros de versão como snapshots (fotos) do estado do projeto, além da referência/caminho para essa foto.

 >> GIT e suas operações locais
A maioria das operações feitas pelo GIT são locais e por isso boa parte das operações são praticamente instantâneas devido à facilidade de acessar arquivos em seu próprio computador.


>> GITHUB
É uma plataforma de hospedagem de código.
- Usado como portfólio (complemento do currículo);
- Usado para estudos;
- Usado por corporações;
- Usado também pela comunidade para atualizações de código aberto.


>> BRANCHS 
Sâo ramificações que podem ser feitas, para que os devs trabalhem separadamente para adicionar um novo recurso ou corrigir um erro, e posteriormente pode-se mesclar a ramificação com a principal (main ou master)
